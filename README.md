# Shopping Cart — HTML/JavaScript

A fully functional shopping cart UI built with vanilla JavaScript. Add products, adjust quantities, remove items, and see the running total update in real time.

## Features

- Product listing with Add to Cart buttons
- Cart sidebar with item list and quantities
- Increase / decrease / remove items
- Live subtotal and total calculation
- Empty cart state handling

## Running

Open `index.html` in any browser — no build step required.

## Tech Stack

HTML5, CSS3, Vanilla JavaScript
